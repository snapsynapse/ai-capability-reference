## Why Watching

Intelligent Internet is an open-source AI agent framework with two main products: **ii-agent** (autonomous AI assistant, Apache 2.0, #5 on Terminal Bench 2, 75.57% GAIA) and **Common Ground** (multi-agent collaborative workspace, launched Feb 2026). ii-agent V1 supports multi-model chat (Gemini 3, Claude, GPT-5.2), BYOK, universal connectors (GitHub/Slack/Gmail/Notion/etc.), browser automation, full-stack development, and image/video generation.

Genuine open-source project with real capabilities and breadth, but traction is well below the tracked self-hosted tools:

| Tool | GitHub Stars | Status |
|------|-------------|--------|
| Ollama | ~165k | Tracked |
| text-generation-webui | ~46k | Tracked |
| LM Studio | proprietary (millions of downloads) | Tracked |
| **ii-agent** | **~3.1k** | **Watching** |
| Common Ground | ~100 | Too early |

At ~2% of Ollama's traction and ~7% of text-generation-webui's, ii-agent does not yet meet the practical inclusion heuristic. No visible funding, team background, or commercial presence. Last tagged GitHub release was v0.4 (Jul 2025); the "V1" framing appears on the website but not as a tagged release.

## Would Add If

- **Traction threshold:** GitHub stars cross ~10k (approaching text-generation-webui territory), indicating meaningful community adoption
- **Commercial signal:** A funded entity, commercial offering, or enterprise tier emerges behind the project
- **Benchmark durability:** Terminal Bench or similar benchmarks show sustained top-5 placement across multiple evaluation cycles, not just one
- **Distribution reach:** ii-agent gets picked up by a major distribution channel (e.g., bundled with a hardware vendor, integrated into a platform, pre-installed in a runtime)
- **Common Ground breakout:** The multi-agent collaborative workspace gains meaningful independent traction (1k+ stars, external adoption)
