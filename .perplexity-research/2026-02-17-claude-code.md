# Claude Code IDE Integrations — February 2026

**Query:** Claude Code IDE integrations February 2026: VS Code extension, JetBrains plugin, web version, iOS availability, latest features
**Model:** sonar-pro
**Date:** 2026-02-17

---

## Answer

- **VS Code extension**: Fully released from beta, native sidebar panel (Spark icon), real-time diffs, plan mode, auto-accept edits, subagents, slash commands. Works with VS Code variants like Cursor and VSCodium.
- **JetBrains plugin**: No direct Claude Code plugin. JetBrains AI Assistant integrates Claude models (Sonnet 4.5, Opus 4.6) for real-time suggestions, but lacks full agentic features.
- **Xcode**: Native integration in Xcode 26.3 via Claude Agent SDK (Feb 2026).
- **Web version**: CLI accessible via browser-compatible setups (npm, Homebrew, winget); no standalone web UI.
- **iOS**: No mobile Claude Code app. GitHub issue #20447 requesting Android/iOS/Web UI remains open.

### Latest Features (Claude Code 2.0+ with Opus 4.6):
- Agent Teams (experimental multi-agent coordination)
- Checkpoints & rewind (auto-saves before changes)
- Subagents (parallel AI threads)
- Hooks & background tasks
- Extensions layer (CLAUDE.md, skills, MCP, plugins)
